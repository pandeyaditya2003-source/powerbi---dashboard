# Teacher Salary Dashboard - Power BI

## Project Overview
This Power BI dashboard provides insights into teacher salary records based on subject, class time, and teacher name. It helps users analyze and compare teacher-related data through interactive visualizations.

## Features
- Teacher-wise salary analysis
- Subject-wise data visualization
- Class timing comparison
- Interactive Power BI charts and filters
- Easy-to-understand dashboard layout

## Tools Used
- Power BI
- Power Query
- Data Modeling
- Data Visualization

## Dashboard Preview
The dashboard displays the count of salary records categorized by:
- Teacher Name
- Subject
- Class Time

## Objective
To create an interactive reporting solution that enables quick analysis of teacher and class-related information for better decision-making.
